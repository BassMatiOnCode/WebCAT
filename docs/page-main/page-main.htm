Unolved Questions





(solved) Margin between MAIN and Page Toolbar

	
The MAIN content container butts up against some margin at the top. The origin of the margin is unclear. Currently, its size is 20px.

	
Solution: It is the top margin of the H1 element, which stretches beyond the top border of MAIN. Giving MAIN a transparent top border of 1 pixel solves the problem.
