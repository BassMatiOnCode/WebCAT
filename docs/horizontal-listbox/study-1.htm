Boxes and CSS

This is a purely CSS-controlled design. The innermost content drives the height of surrounding boxes, except the outermost box (this keeps the layout stable while the scrollbar is shown or hidden. The item-container provides the horizontal scroll bar, triggered by a :hover condition.

Hover the mouse pointer over the item container. Note that the vertical position of the test content below does not change. On a touch-screen, tap the item container to show the scroll-bar, then push the content to the left or right. Tap outside to hide the scroll-bar.
