Content Height Observer

Fit OBJECT height to hosted content

Introduction

An OBJECT element, hosting image content, automatically adapts its size to fit the content. Unfortunately, non-image content such as text/html does not resize its host, instead, scrollbars are added. To behave like image content, an observer pattern is used.
