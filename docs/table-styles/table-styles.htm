Vertical Cell Text

Vertical cell text is used in tables with tall but narrow columns, or if the number of columns is large:

The demo above demonstrates a number of options. For example you can change the text orientation and alignment. For this purpose, decorate the TR element with the class-name vertical and some optional ones:

TH elements are considered as row header cells. Their text orientation always remains horizontal. However, their vertical alignment can be changed with the class names th-top, th-middle and th-bottom.

The alignment of data cells can be adjusted with the class names td-left, td-center or td-right.

These classnames also works on any row in the table body, i. e. in the middle if the table:

Using the Component

The component consists of a single file:

  • vertical-cell-text.css

For detailed notes on how to import these files into a document or aggregation script, refer to WebCAT Component Anatomy.

Decorate the TR element with the class name vertical to change the text orientation of TD cells in that row. Add additional class names, as defined in the CSS file, to obtain additional orientation and alignment effects.
