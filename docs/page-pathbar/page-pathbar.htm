Added Scroll-Box Class



By adding the horizontal scroll-box class names to the path-bar component and adding the scroll-box component, a dynamic horizontal scrollbar was added to the path-bar. The path-bar now expands down to show a horizontal scrollbar if its content does not fit into the available width.



This required changes in the CSS z-index and position properties of components. The first attempt to raise the z-index value. That failed because the path-bar then covered the sitemap navigation panel. Instead, the z-index of the following toolbars and the main content container was lowered to negative values.
