Math Equation Fragment

Math equations as re-usable fragments

Introduction

This component supports re-usable math equation fragements and provides styles for math equations and shows a title string if the fragment is shown in its own window.

Using the Component

The component consists of two files:

  • math-equation-fragment.css
  • math-equation-fragment.js

These files are used in the HTML fragment that contains the equation.

In a host document, use an HTML OBJECT element to import the fragment:

For detailed notes on how to import these files into a document or aggregation script, refer to WebCAT Component Anatomy.

More information on how to use the module can be found in the examples section.
