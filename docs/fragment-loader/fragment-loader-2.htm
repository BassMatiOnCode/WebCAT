HTML Fragment Loader

Events

The fragment loader raises the following events:

fragment-loader-root-containers-finished

Dispatched to the document element, raised in processRootContainers(). Announces that the fragment loader has processed all rootContainers passed in the event.details.

fragment-loader-root-container-finished

Dispatched to each element passed in rootContainers to processRootContainers(). The event.details provide a reference to the rootContainer that has been processed.

fragment-loader-container-finished

Dispatched to the container element passed to loadFragments(). NOTE that this event is raised on each root container passed to processRootContainers(), as well as the containers of all nested HTML fragments, because fragment loading is a recursive procedure, and each HTML fragment can be the root node of an entire tree of fragments.

The event.detail object provides a reference to the container element and an array of all HTML fragment anchors that have been processed.

fragment-loader-selecting-content

Dispatched to anchor element passed to loadFragment() if the HTML fragment anchor has a data-select attribute. Announces that the fragment content has been loaded into a buffer, and a portion of that content is about to be selected for injection into the host document. This event is used to grab additional information from the fragment content that will be inaccessible afterwards.

fragment-loader-injecting-content

Dispatched to the HTML fragment anchor passed to injectContent(). Announces that the buffer content is about to be injected into the host document. The anchor element is still a member of the host document, but might be replaced and thereby removed from the document. This is the last chance to tweak the content that is about to be injected.

fragment-loader-content-injected

Dispatched to the HTML fragment anchor passed to injectContent(), after the content has been injected into the host document. The anchor element might be no longer be a member of the host document, for example if the injected content has replaced the anchor.
