Ways to Inject HTML

This study examines the ways to inject HTML code into a host document.

InnerHTML

Assigning an HTML string to element.innerHTML parses the text and creates DOM objects, which replace the entire element's content.

Scripts are not executed. Styles are evaluated and applied.

Using DOMParser

This container uses a DOMParser to create an inert HTML document object. Scripts are not executed, images are not loaded, events are not fired. Its BODY containts the resulting DOM tree, which is then moved to the target container.

The input string can include declarative shadow roots.

range.createContextualFragment( )

This method uses range.createContextualFragment( ) to create a document fragment object. This method does parse JavaScript elements in the input string correctly. The script is executed when the fragement is injected into the target container in the host document.
