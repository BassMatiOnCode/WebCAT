Page Header

Creates a page header with logo and title images.

The page header is one of the horizontal panels at the top of the document that are inserted into an existing DOM tree. Specifically, the page content below can be shifted down by injecting elements above. This is countermeasured by top or bottom margin of elements. Before injection, there must be sufficient margin to accommodate the injected panels. These margins are diminished by the height of the inserted panels, so that the elements below remain in their position (i. e. their offsetTop does not change).

By default, document.body provied enough marginTop to accommodate the page header, toolbar and breadcrumbs panels. If their component scripts are imported in opposite order, no module parameters have to be supplied.

NOTE that the body margin is defined in elements.css, which resides in the site root folder. This stylesheet link must be included before these panels.
