Literature Reference List

Note that the listerature reference list can be a separate document.

The component consists of two files:

  • literature-reference-list.css
  • literature-reference-list.js

For detailed notes on how to import these files into a document or aggregation files, refer to WebCAT Component Anatomy.

To successfully use the literature-reference-list component, some requirements must be met:

  • The list must be decorated with the class name literature-reference-list
  • Each title entry in the list must have a unique numeric ID
