Study 8 Sender

Using Local Storage

Send Open Receiver

This study examines how to persist and relay messages with the LocalStorage API. If a window changes the local storage, other windows can receive storage events and react accordingly. This mechanism is exploited in the math fragment list window (receiver.htm).

Usage: Enter some text in into the input box and press Enter or press the "Send" button. The input will be stored in the local storage, and appear in the receiver window at the bottom, if it is open.

Press the "Open Receiver" button to open the receiver window.

Shift-click the "Send" button to open the reveiver in a new window. Ctrl-click the "Send" button to open the receiver in a new browser tab.
