Study 8 Receiver

Upon start, the receiver retrieves the studyStorageKey item from local storage, extracts the strings and creats P elements in the textContainer with them.

The user can select an element and delete it. The delete button handler updates the local studyStorageKey item with the new (shortened) list.

The receiver listens for storage events with a matching studyStorageKey. These can only be generated by the sender page (study-08.htm in this case) that appended a new item to the list. The storage event handler picks that last item from the list, generates a new P element and appends it to the textContainer DIV.
