SPA Content Component

Creating Content Documents

Use the SPA content component(s) to easily create SPA content documents.

Usage

Using a Barebone Template

  1. Copy a preconfigured template like path/to/spa-article-template.htm to your website folder and rename it appropriatly.
  2. Add a corresponding entry in a suitable toc.htm file so that it is visible in the table of contents.

Create a Content Document from Scratch

  1. Create an empty HTML file with the usual META elements in the HEAD section.
  2. Add a redirection script as first element in the head section. Make sure to reference the correct SPA host document, /index.htm or another appropriate host document.
  3. Add a BASE target="_top" element.
  4. Add a LINK rel="stylesheet" href="path/to/spa-iframe-content.css" element.
  5. Add a SCRIPT type="module" src="path/to/spa-iframe-content.js" element.

If your SPA host document uses a different container such as a DIV or MAIN element, use spa-div-content.css and spa-div-content.js instead.
