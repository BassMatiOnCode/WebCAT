Using the Component

The component consists of two files:

  • math.css
  • math.js
  • math-text.css

The files math.css and math.js support MathML block and inline elements. math-text.css is for writing math in HTML, which is sometimes simpler than using MathML elements. It ensures the same font and similar styles as in MathML.

For detailed notes on how to import these files into a document or aggregation script, refer to WebCAT Component Anatomy.

More information on how to use the module can be found in the examples section.
