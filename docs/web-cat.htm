WebCAT

A web content authoring toolbox

Introduction

Typical WebCAT components are visible, like the page header, footer, path bar, toolbar, tool buttons, navigation panel. Other components are invisible and provide functionality, such as the collapsible structures component, CSS counters, sitemap navigator, page scrolling, page templates and other components. Then there are programming support components, such as the XML namespace or the utility components.
